# Calculator Application

## Description

This is a GUI-based Calculator application developed using Python and Tkinter. It performs basic arithmetic operations such as addition, subtraction, multiplication, and division through an easy-to-use graphical interface.

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Decimal calculations
- Clear button
- Error handling

## Technologies Used

- Python 3
- Tkinter

## How to Run

```bash
python calculator.py
```
