# Password Generator

A simple Python-based Password Generator that creates random passwords based on the user's preferred length and complexity.

## Features

- User-defined password length
- Three password complexity levels:
  - Letters Only
  - Letters + Numbers
  - Letters + Numbers + Special Characters
- Random password generation
- Beginner-friendly command-line interface

## Technologies Used

- Python 3
- random module
- string module

## How to Run

1. Clone the repository.

```bash
git clone https://github.com/your-username/CODSOFT_TASKS.git
```

2. Navigate to the project folder.

```bash
cd Task3_Password_Generator
```

3. Run the program.

```bash
python password_generator.py
```
