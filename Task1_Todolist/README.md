# To-Do List Application (GUI)

## Description

The **To-Do List Application** is a Python-based desktop application developed using the **Tkinter** library. It helps users organize their daily tasks through a simple graphical user interface (GUI). Users can easily add, view, update, and delete tasks, making task management more efficient and user-friendly.

## Features

* Add new tasks
* View all tasks in a list
* Update existing tasks
* Delete selected tasks
* Easy-to-use graphical interface
* Instant task management
* Beginner-friendly design

## Technologies Used

* Python 3
* Tkinter (GUI Library)

## How to Run

1. Make sure Python 3 is installed.
2. Open the project folder.
3. Run the following command:

```bash
python todolist.py
```

4. The To-Do List window will open.
5. Use the buttons to add, update, or delete tasks.
