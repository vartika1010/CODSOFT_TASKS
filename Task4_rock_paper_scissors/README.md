# Rock Paper Scissors Game

## Description

This is a simple Rock Paper Scissors game built using Python and Tkinter as part of the CodSoft Python Programming Internship (Task 4). The game allows the user to play against the computer through an interactive graphical user interface.

## Features

* User-friendly GUI built with Tkinter
* Random computer choice
* Winner determination based on game rules
* Live score tracking
* Play Again button
* Reset Score button
* Modern pink and purple themed interface

## Technologies Used

* Python 3
* Tkinter
* Random module

## How to Run

1. Make sure Python 3 is installed.
2. Download or clone this repository.
3. Navigate to the project folder.
4. Run:

```bash
python rock_paper_scissors.py
```

## Game Rules

* Rock beats Scissors.
* Scissors beat Paper.
* Paper beats Rock.
* If both players choose the same option, the round ends in a tie.
